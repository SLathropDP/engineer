--- title: List ---

List

Configure a SharePoint list.

Access

Access the List API using the getByTitle() or getById() method of the Lists API.

Examples

{% highlight js %} up(engineer) { // Update list engineer.web.lists.getByTitle('My List').update({ Description: 'New list description', }); } {% endhighlight %}

Properties

contentTypes

Provides access to the ContentTypes API.

fields

Provides access to the Fields API.

views

Provides access to the Views API.

Methods

update(options)

Update the list.

Parameters

options
Updated list configuration (Object, required)

Properties of options

See Microsoft’s List resource documentation for a listing of all possible properties that can be passed to the REST API.

Examples

{% highlight js %} up(engineer) { // Update list engineer.web.lists.getByTitle('My List').update({ Description: 'My new list description', }); } {% endhighlight %}

delete()

Delete the list.

Examples

{% highlight js %} down(engineer) { // Delete list engineer.web.lists.getByTitle('My List').delete(); } {% endhighlight %}
