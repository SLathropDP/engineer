--- title: ContentType ---

ContentType

Configure a SharePoint content type.

Access

Access the ContentType API using the getByName() or getById() method of the ContentTypes API.

Examples

{% highlight js %} up(engineer) { // Update content type on web engineer.web.contentTypes.getByName('My Content Type').update({ Description: 'This is a new description', }); // Update content type on list engineer.web.lists.getByTitle('My List').contentTypes.getByName('My Content Type').update({ Description: 'This is a new description', }); } {% endhighlight %}

Properties

fieldLinks

Provides access to the FieldLinks API.

Methods

id()

Get the content type ID or GUID from Engineer’s content type ID manifest.

Returns

String
The ID or GUID of the content type, or null if no ID was found

Examples

{% highlight js %} up(engineer) { // Create parent content type engineer.web.contentTypes.add('New Parent'); // Create child content type, getting parent's ID from the manifest engineer.web.contentTypes.add({ ParentContentTypeId: engineer.web.contentTypes.getByName('New Parent').id(), Name: 'New Child Content Type', }); } {% endhighlight %}

update(options)

Update the content type.

Parameters

options
Updated content type configuration (Object, required)

Properties of options

Description
A description of the content type (String, optional)
Group
The group name under which this content type will appear in content type galleries (String, optional)

Examples

{% highlight js %} up(engineer) { // Update content type on web engineer.web.contentTypes.getByName('My Content Type').update({ Description: 'This is a new description', Group: 'My Group', }); } {% endhighlight %}

delete()

Delete the content type.

Examples

{% highlight js %} down(engineer) { // Delete content type from web engineer.web.contentTypes.getByName('My Content Type').delete(); } {% endhighlight %}
