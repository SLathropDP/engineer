--- title: View ---

View

Configure a SharePoint view in a list.

Access

Access the View API using the getByTitle() or getById() method of the Views API.

Examples

{% highlight js %} up(engineer) { // Update view engineer.web.lists.getByTitle('My List').views.getByTitle('All Items').update({ RowLimit: 100, }); } {% endhighlight %}

Properties

viewFields

Provides access to the ViewFields API.

Methods

update(options)

Update the view.

Parameters

options
Updated view configuration (Object, required)

Properties of options

See Microsoft’s View resource documentation for a listing of all possible properties that can be passed to the REST API.

Examples

{% highlight js %} up(engineer) { // Update view engineer.web.lists.getByTitle('My List').views.getByTitle('All Items').update({ RowLimit: 100, }); } {% endhighlight %}

delete()

Delete the view.

Examples

{% highlight js %} down(engineer) { // Delete view engineer.web.lists.getByTitle('My List').views.getByTitle('My View').delete(); } {% endhighlight %}
