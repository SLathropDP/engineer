engineer init

Initialize the current working directory as an Engineer project folder.

The init command creates .gitignore and env.js files if they do not already exist in the current working directory.

Important Security Consideration

If you are using source control to track migration changes, we highly recommend ignoring the env.js from any code repositories in order to prevent sensitive authentication information from being shared. If you are using git source control, the .gitignore file will take care of this for you.

Examples

{% highlight sh %} engineer init {% endhighlight %}
