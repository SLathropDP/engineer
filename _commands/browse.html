--- layout: commands title: browse ---

engineer browse [list]

Open SharePoint home page or list in web browser.

Options

  • list — if no list name is provided, the SharePoint home page will be shown; if a list name is provided, the browser will navigate directly to the default list view for the given list; a value of migrations or manifest will browse Engineer’s hidden tracking lists

Examples

{% highlight sh %} # Browse to SharePoint home page engineer browse # Browse to default view for "My List" engineer browse "My List" # Browse to Engineer's migrations list engineer browse migrations # Browse to Engineer's manifest list engineer browse manifest {% endhighlight %}
