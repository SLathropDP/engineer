engineer uninstall

Uninstall Engineer’s hidden tracking lists from SharePoint.

The uninstall command deletes all lists deployed by the install command from the target SharePoint environment. Note that after uninstalling, the data from the migrations and manifest lists will be lost. Be sure to roll back all active migrations before uninstalling to avoid errors in future migrations and rollbacks.

Examples

{% highlight sh %} engineer uninstall {% endhighlight %}
